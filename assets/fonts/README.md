# Embedded font

`NotoSansCJKsc-Regular.otf` is the Simplified Chinese Regular face from the
[Noto CJK repository](https://github.com/notofonts/noto-cjk), downloaded from:

`https://raw.githubusercontent.com/notofonts/noto-cjk/main/Sans/OTF/SimplifiedChinese/NotoSansCJKsc-Regular.otf`

It is distributed under the SIL Open Font License 1.1 in `OFL.txt`. The same
font bytes are used for Go layout measurement and resvg text rendering.
